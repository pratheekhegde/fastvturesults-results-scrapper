# Fastvturesults Scrapper

**The code is very dirty, it is not optimized!**

### Steps

* run `npm install`
* edit your **USN** inside `server.js`
* run `node server.js`
* `output.json` will contain all your results.
* You will have all your results from **fastvturesults.com**
